Governance Architecture

Core Governance Layers:
1. Identity Governance
2. Resource Governance
3. Cost Governance
4. Compliance Governance

Identity Governance:
- Role design
- Permission reviews

Resource Governance:
- Naming standards
- Tagging rules
- Allowed SKUs and regions

Cost Governance:
- Budget alerts
- Cost tracking
- Reserved capacity decisions

Compliance Governance:
- Azure Policy
- AWS Config
- CIS baseline enforcement
