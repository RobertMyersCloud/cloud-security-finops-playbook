# FinOps KPIs

These KPIs map directly to the FinOps Foundation and enterprise reporting.

---

# Cost Visibility KPIs
- Allocation Coverage %
- Unallocated Spend
- Cost per App / Team / Owner
- Top N Cost Drivers

# Efficiency KPIs
- Rightsizing Coverage %
- Idle Resource Cost
- Spot Usage %
- RI/SP Utilization %

# Forecasting KPIs
- Forecast Accuracy %
- Budget Variance %
- MoM Spend Trend

# Architecture KPIs
- Storage Lifecycle %
- Serverless Adoption %
- Container Efficiency

# Business Value KPIs
- Unit Economics (Cost Per X)
- Cost-to-Revenue Ratio
- Feature/Workload Value
