# FinOps Maturity Model (Crawl → Walk → Run)

This follows the official FinOps Foundation maturity framework.

---

# Crawl
- Basic dashboards  
- Unstructured tagging  
- Initial forecasting  
- Some cost visibility  

# Walk
- Tagging standards  
- RI/SP planning  
- Optimization cycles  
- Engineering accountability  

# Run
- Automated governance  
- ML-based forecasting  
- Comprehensive dashboards  
- Unit economics  
- Cost feedback loops
