# Power BI Data Modeling for Cloud Analytics

## Data Modeling Principles
- Star schema
- Clean dimensional modeling
- Use Date, Region, App, Owner dimensions

## Key Tables
- CostUsage (fact)
- Resources (dim)
- Tags (dim)
- Accounts/Subscriptions (dim)
- SecurityFindings (fact)
- Incidents (fact)

## Best Practices
- Avoid bi-directional relationships
- Use surrogate keys
- Normalize tag schema
