Post-Incident Lessons Learned

Key Review Items:
- What failed or was missing in detection?
- Were guardrails insufficient?
- Did IAM permissions allow excessive access?
- Was logging enabled everywhere?

Improvements:
- Add new detections.
- Tighten IAM roles.
- Enforce MFA more broadly.
- Add automated remediation rules.
- Improve incident documentation.

Outcome:
- Stronger security posture
- Reduced response time
- Better cloud governance maturity
